# form styles

## Customizing input styles

1. By setting border to 0, we can get rid of some of the default styling on our input
2. Then even out all the input elements by setting the padding property
```css
input {
  border: 0;
  padding: 10px;
  font-size: 18px;
}
```

## Styling submit button

```css
input[type="submit"] {
    background: red;
    color: white;
}
```
