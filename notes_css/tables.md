# Tables

## Removing top border from html table cells
- http://stackoverflow.com/questions/24290835/removing-top-border-from-html-table-cells

```css
.spec_table td {
    margin-left: -20px;
    font-size: .9em;
    line-height: 1.1em;
    border-top: none !important;
}
```

## Center the table data in a th/td

thead tr th,
.btn_box {
  text-align: center;
}
