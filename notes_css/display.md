# display property

### block
- **Stretches the whole width**
- **Can have line breaks** before and after
- like Android's match_parent
```
    + <h1>, <h2>, ...
    + <p>
    + <ul>, <ol>, <li>
    + <div>
```

### inline
- **no line breaks**
- Only takes up as much of the screen as need
- like Android's wrap_content
```
    + <a>
    + <img>
    + <input>
    + <span>
```
